【Android教学】【已解决】某书郎学生平板安装三方应用教程

阅读时长：10 分钟 文章编号:#001 操作前提：熟悉 windows 基本操作方式、熟悉键盘打字

现在很多家长都给孩子买了一个学生平板，但是有很多孩子都不知道该如何才能在学生平板上安装三方应用，本文章就
教您如何在学生平板上安装三方应用。
准备材料：windows11 系统（推荐专业版）、adb 命令工具合集、一台学生平板（系统版本推荐 Android7.1 级以下，更方便安装)
这里以某书郎举例子。在打开之前还请您知晓管控密码。如果您不晓得请您在家长输密码之前打开录屏软件录制下管控密码。

学习机：设置–关于设备，连续多次点击版本号进入开发者模式，再次点击 "开发者选项"–键入管控密码 ---- 打开 usb 调试
连接计算机。
计算机：打开 windows PowerShell（以管理员身份运行），cd 至 adb 命令工具安装目录，再次键入 “.\adb shell”
学习机：当您的学习机提示是否接受计算机的调试，选择 " 同意 / 确定 “
现在您可以使用 adb 命令安装应用了
adb 用法：
db install <APK 路径>：安装应用。
adb uninstall <包名>：卸载应用。
adb uninstall -k <包名>：卸载应用但保留数据和缓存文件。
adb shell am start -n <包名>/<Activity 名 >：启动指定应用的指定 Activity。
adb shell am start -a -d <DATA_URI>：根据 Intent 启动应用。
adb shell：进入设备的 Unix shell。
adb reboot：重启设备。
adb pull <远程路径> < 本地路径 >：从设备复制文件到本地。
adb push <本地路径> < 远程路径 >：将文件推送到设备。
